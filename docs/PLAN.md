# JLPT RPG 기획서

> 픽셀 RPG 던전 컨셉의 JLPT 일본어 단어 암기 웹앱

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | JLPT RPG |
| 한 줄 소개 | 외운 단어 수가 곧 처치한 몬스터가 되는 일본어 단어 학습 RPG |
| 플랫폼 | 웹 (모바일 우선 반응형, 데스크탑 대응) |
| 배포 | Vercel |

### 1.1 목적

JLPT(N5~N1) 시험 대비 한자 단어 학습을 RPG 게임처럼 만들어, **외운 단어 수가 곧 처치한 몬스터 수**가 되는 시각적 동기부여 시스템을 제공한다.

### 1.2 타겟

- JLPT 시험을 준비하는 한국어권 학습자
- 단어 암기를 지루해하는 유저
- 모바일로 짬짬이 공부하고 싶은 유저

### 1.3 핵심 차별점

- **RPG 게임 시스템** — 단어 외움 = 몬스터 처치, 누적 카운트로 모험가 등급 상승
- **한자 어원 학습** — 단순 암기가 아니라 한자 분해·유래까지 카드 뒷면에 제공
- **픽셀 도트 캐릭터** — 학습 중 화면에 캐릭터가 함께 있고, 단어를 외울 때 attack 모션 + 이펙트로 시각적 보상

## 2. 기능 명세

### 2.1 인증

- 이메일/비밀번호 회원가입·로그인
- **자동 로그인** — 세션 유지, 재방문 시 자동 진입
- **네이버 간편 로그인** (OAuth) — Vercel Serverless Function 으로 콜백 처리

### 2.2 학습 (메인 기능)

- 단어 카드:
  - **앞면**: 한자 + 히라가나 읽기 (히라가나-only 단어는 히라가나만 크게)
  - **뒷면**: 한국어 뜻 + 한자 어원/분리 설명 + 일본어 예문 + 한국어 번역
  - 탭하여 뒤집기 (3D 플립 애니메이션)

- **4가지 액션 버튼** (2x2 그리드):

| 액션 | 동작 | 처치 카운트 |
|---|---|---|
| 완벽히 외움 (👑) | mastery = mastered | +1 |
| 외운 것 같아요 (✓) | mastery = probably, 가끔 재등장 | 미반영 |
| 다시 보기 (🔖) | flagged 토글, 북마크 보관함에 추가 | 미반영 |
| 다음 (→) | 상태 변경 없이 스킵 | 미반영 |

- 액션 후: 캐릭터 attack 모션 + 카드 측 이펙트 (설정으로 off 가능)
- "기억 상태" 옆 ⓘ 아이콘으로 각 상태 설명 모달

### 2.3 단어 노출 순서 옵션

- **랜덤 / 순서대로** 선택
- **외운 단어 제외** 체크박스 (mastered 만 제외)
- **"외운 것 같아요" 재등장 주기** 슬라이더 (기본 8장에 1번, 1~20 조절)
- **"다시 보기" 가중치** 슬라이더 (기본 3배, 1~10 조절)

### 2.4 다시 보기 시스템 (이중 운영)

flagged 된 단어는 **두 방식 모두** 적용:
- **일반 학습 모드**: 가중치(`review_mix_weight`)만큼 더 자주 등장
- **다시 보기 모드**: 별도 진입 시 flagged 단어만 모아서 학습

### 2.5 캐릭터

- **4종**: 전사 / 마법사 / 궁수 / 소환사 — 학습 중 좌측 하단 배치
- **언제든 변경 가능** (학습 화면에서도)
- 모션 상태:
  - **idle**: 살짝 씰룩씰룩 (위아래 1~2px)
  - **attack**: 액션 버튼 누를 때 앞으로 휘두름

### 2.6 공격 이펙트 (캐릭터별 차별화)

| 캐릭터 | 이펙트 | 컬러 |
|---|---|---|
| ⚔ 전사 | 대각선 슬래시 | 흰색/은색 |
| 🔮 마법사 | 파티클 폭발 | 보라 |
| 🏹 궁수 | 화살 트레일 | 녹색 |
| 👻 소환사 | 확장 링 | 노랑/주황 |

설정 토글 (3단계):
- 캐릭터 모션 on/off
- 카드 이펙트 (슬래시/파티클 등) on/off
- 사운드 on/off

### 2.7 던전 시스템

JLPT 5개 레벨을 던전 테마로 매핑:

| 레벨 | 던전 이름 | 컬러 톤 |
|---|---|---|
| N5 | 시작의 숲 | 녹색 |
| N4 | 동굴 | 갈색 |
| N3 | 고대 유적 | 양피지/주황 |
| N2 | 화산 | 빨강·주황 |
| N1 | 천공의 신전 | 보라·금색 |

- **던전별 처치 카운트 분리** — "N3 던전에서 47마리 처치" 식으로 각 던전 진척도가 따로 표시됨
- 홈 화면에서 각 던전 진입 가능 + 자신의 처치 수 미리보기

### 2.8 등급 시스템

누적 처치 수 (전체 던전 합산) 기준 5단계 등급:

| 처치 수 | 등급 (한·일 병기) |
|---|---|
| 0~49 | 견습 모험가 (見習い) |
| 50~199 | 모험가 (冒険者) |
| 200~499 | 베테랑 (ベテラン) |
| 500~999 | 정예 (エリート) |
| 1000+ | 전설 (伝説) |

- 등급명 일본어 병기 → 그 자체로 학습 콘텐츠
- 등급 상승 시 작은 레벨업 이펙트 + 칭호 뱃지 갱신

### 2.9 나만의 단어장

- 사용자가 직접 단어 덱을 만들 수 있는 커스텀 기능
- 입력 방식:
  - 폼으로 1개씩 직접 입력
  - **CSV 파일 업로드** — 포맷: `한자, 읽기, 뜻, 어원, 품사`
  - (선택) 예문은 별도 CSV 또는 단어별 추가 입력
- 커스텀 덱도 공식 덱과 동일한 학습 흐름 적용
- 등급/처치 카운트에는 모두 합산

### 2.10 설정 화면

- 노출 순서 / 외운 단어 제외 / 재등장 주기 / 다시보기 가중치
- 이펙트 토글 3종
- 캐릭터 변경
- 닉네임 변경
- 계정 (이메일 확인, 로그아웃)

```json

  "order": "random",
```

## 7. 결정 사항 요약

- 단어 외움 단계는 **2단계** (완벽히 외움 / 외운 것 같아요) + 다시 보기는 **별도 플래그**
- 처치 카운트는 **던전별로 분리** 표시
- 캐릭터는 **언제든 변경 가능**
- 다시 보기는 **일반 학습 가중치 + 별도 모드** 둘 다 운영
- 노출 주기·가중치는 **유저가 설정에서 조절 가능**
- 모든 이펙트는 **on/off 토글 제공**
- 모바일 **완벽 반응형** 우선
- 첫 화면은 시안의 **양피지 카드 + 픽셀 캐릭터** 컨셉
